# DELAY

## Sintassi

  ```

  ```

## Parametri
|Parametro            | Descrizione                                                                     |                
|---------------------|---------------------------------------------------------------------------------|
| **valore**          | costante o variabile integer o double. Tempo di ritardo espresso in secondi.    |
