# RESETOUT

## Sintassi

  ```
  RESETOUT	nomeoutput
  ```

## Parametri
|Parametro                | Descrizione                                                 |                
|-------------------------|-------------------------------------------------------------|
| **nomeoutput**          | nome di dispositivo tipo uscita digitale                    |               

## Descrizione
Disattiva (pone ad OFF) l'uscita **nomeoutput**.
