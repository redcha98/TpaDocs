# RESETSTATE

## Sintassi

  ```
  RESETSTATE	nomeboolean
  ```

## Parametri
|Parametro                | Descrizione                                                 |                
|-------------------------|-------------------------------------------------------------|
| **nomeboolean**         | nome di variabile booleana                                  |    

## Descrizione
Disattiva (pone ad OFF) la variabile **nomeboolean**.
