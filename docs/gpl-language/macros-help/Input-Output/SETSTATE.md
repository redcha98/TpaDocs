# SETSTATE

## Sintassi

  ```
  SETSTATE	nomeboolean
  ```

## Parametri
|Parametro                | Descrizione                                                 |                
|-------------------------|-------------------------------------------------------------|
| **nomeboolean**         | nome di variabile booleana                                  |    

## Descrizione
Attiva (pone ad ON) la variabile **nomeboolean**.
