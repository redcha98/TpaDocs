# GETREALTIMECOUNT

## Sintassi

  ```
GETREALTIMECOUNT nomevar
  ```

## Parametri
|Parametro              | Descrizione                                           |                
|-----------------------|-------------------------------------------------------|
| **nomevar**           | variabile integer                                     |  


## Descrizione
Ritorna nella variabile **nomevar** il numero di real-time di gestione degli assi eseguiti dall'ultima inizializzazione del controllo numerico. 


Vedere anche [GETREALTIME](GETREALTIME.md).
