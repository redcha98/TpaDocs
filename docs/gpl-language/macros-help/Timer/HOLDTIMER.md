# HOLDTIMER

## Sintassi

  ```
  HOLDTIMER	nometimer
  ```

## Parametri
|Parametro                | Descrizione                                 |                
|-------------------------|---------------------------------------------|
| **nometimer**           | nome di variabile tipo timer                |               

## Descrizione
Blocca l'aggiornamento del timer **nometimer**.

Vedere anche [STARTTIMER](STARTTIMER.md) e [SETTIMER](SETTIMER.md)
