# STOPMOV

## Sintassi

  ```
STOPMOV asse
  ```

## Parametri
|Parametro       | Descrizione                     |                
|----------------|---------------------------------|
| **asse**       | nome di dispositivo tipo asse   |         

## Descrizione
Ferma il movimento dell'**asse** (o del canale di interpolazione identificato da **asse**).
