# WAITSTILL

## Sintassi

  ```
 WAITSTILL asse1 [, ..., asse6] 
  ```

## Parametri
|Parametro                 | Descrizione                     |                
|--------------------------|---------------------------------|
| **asse1...[...asse6]**   | nomi di dispositivi tipo asse   |         

## Descrizione
Attesa per tutti gli **assi** (1รท6) specificati del termine del movimento (stato Quota).
